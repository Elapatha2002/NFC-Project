Put company logo image files in this folder (e.g. abc-logo.png).

In the admin panel's "Logo" field you can then enter just the file name
(abc-logo.png) and it will be served from /logos/abc-logo.png.

Alternatively, enter a full image URL (https://...) in the Logo field and
you don't need to place a file here at all (e.g. a Firebase Storage URL).
